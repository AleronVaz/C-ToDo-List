# C-ToDo-List

A simple **menu-driven To-Do List application** written in C using file handling.

## Initial Features
- Basic **To-Do list created using file handling**.
- Menu-driven interface using **switch-case** and number options.
- Features implemented so far:
  - **Add Task**
  - **Exit**
  - **View Task**
